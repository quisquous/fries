# Meteor Fries

A smart package for use with [Meteor](http://meteor.com/) and the [Meteorite](http://oortcloud.github.io/meteorite/) package manager via [Atmosphere](https://atmosphere.meteor.com/).

[Fries](http://jaunesarmiento.me/fries/) is a mobile development framework for Android apps, inspired by [Ratchet](http://maker.github.io/ratchet/).

## How to install

1. `npm install -g meteorite`
2. `mrt add fries`
